Dungeon Shift starten
=====================

Open NIET game.js. Dat geeft op Windows een Script Host foutmelding.

Gebruik een van deze opties:

1. Dubbelklik op START_GAME.bat
2. Of dubbelklik op index.html

Als je het spel naar vrienden stuurt:

1. Zip de hele map:
   ik-wil-een-spel-gaan-ontwerpen

2. Stuur die zip naar je vriend.

3. Je vriend pakt de zip uit.

4. Je vriend dubbelklikt op START_GAME.bat of index.html.

Belangrijk:
- De map assets moet naast index.html blijven staan.
- De map scripts hoeft niet gebruikt te worden om te spelen, maar mag gewoon in de zip blijven.
- Open game.js niet direct. Dat bestand is alleen code voor de browsergame.
